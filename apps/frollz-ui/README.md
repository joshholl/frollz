# Frollz UI

## Description
Frontend application for the Frollz film roll tracking system built with Vue.js, TypeScript, and Tailwind CSS.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run dev

# build for production
$ npm run build

# preview production build
$ npm run preview
```

## Environment Variables
Create a `.env` file in the root directory:

```
VITE_API_URL=http://localhost:3000
```

## Features
- Dashboard with overview statistics
- Film format management 
- Stock management
- Roll tracking with state management
- Responsive design with Tailwind CSS
- TypeScript for type safety
